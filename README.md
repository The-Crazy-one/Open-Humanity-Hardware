# Open-Humanity-Hardware
Modulare Lösungen für globale Krisen. Von der Straße bis ins Katastrophengebiet.

📂 Unsere aktuellen Projekte
🏗️ Supporting_Center
Faltbare Raumsysteme für die Katastrophenhilfe.
Kern: Skalierung vom Kinderreisebett zum Wohncontainer.
Vorteil: Extreme Platzersparnis bei Transport und Lagerung.
Einsatz: Flüchtlingshilfe (z.B. Ahrtal, Afghanistan), mobile Kliniken, autarke Verwaltung.
👉 Zum Projekt-Details & Bauplänen
🛒 Wohntrolley
Mobiler Lebensraum für urbane Würde.
Kern: Ein modifizierter Trolley mit integriertem Sitz, Schlafzelt und Duschkabine.
Vorteil: Gibt Obdachlosen einen Platz auf Augenhöhe und schützt die Privatsphäre.
Einsatz: Hilfe für Menschen ohne festen Wohnsitz, Micro-Camping & Vanlife.
